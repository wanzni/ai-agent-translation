## 部署说明

### 1. 环境要求
- MySQL 8.0+
- 字符集: utf8mb4
- 存储引擎: InnoDB
- 内存: 建议8GB+

### 2. 部署步骤
1. 创建数据库和用户
2. 执行建表脚本: `01_create_tables.sql`
3. 执行初始化数据: `02_init_data.sql`
4. 执行索引优化: `03_create_indexes.sql`
5. 配置应用连接参数

### 3. 配置建议
```sql
-- 推荐配置参数
innodb_buffer_pool_size = 70% of RAM
innodb_log_file_size = 256M
innodb_log_buffer_size = 16M
max_connections = 1000
query_cache_size = 128M
```
